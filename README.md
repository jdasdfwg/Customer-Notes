# Customer Notes

A sleek, clean note-taking app for managing customer notes. Organize notes into folders by customer name with an intuitive three-panel interface.

## Features

- **Folder Management** — Create folders for each customer, rename or delete them
- **Note Taking** — Create, edit, and delete notes within customer folders
- **Search** — Quickly find notes by title or content
- **Auto-Save** — Notes save automatically as you type (localStorage)
- **Clean UI** — Modern three-panel layout inspired by Apple Notes

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js 14** — React framework
- **Tailwind CSS** — Utility-first styling
- **TypeScript** — Type safety
- **localStorage** — Client-side persistence
